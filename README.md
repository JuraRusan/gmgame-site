1. Установить node 16, проще всего использовать nvm
2. Установить пакеты
```bash
npm i
```
3. Собрать фронт
    3.1 Перейти в каталог фрона
    ```bash
    cd idm-web/
    ```
    3.2 Установить пакеты
    ```bash
    mpm i
    ``` 
    3.3 Собрать
    ```bash
    npm run build
    ``` 

    Для просто запуска использовать
    ```bash
    npm run start
    ```
